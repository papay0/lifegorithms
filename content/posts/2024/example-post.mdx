## Technical Stack

For the technically curious, here's what powers this blog:

- **Next.js 16**: React framework with excellent SSR/SSG support
- **MDX**: Markdown with React components

## Features to Explore

### Draft Mode

Posts can be marked as drafts and won't be visible to the public until published. This allows me to work on articles over time before sharing them.

### Tags & Categories

Each post can have multiple tags, making it easier to find related content. You'll see tags displayed on both the homepage and individual post pages.

### SEO Optimization

All posts are optimized for search engines with proper meta tags, Open Graph, and Twitter Cards.
